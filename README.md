# JP-HK Travel Vocabulary SRS

An interactive bilingual vocabulary trainer designed for travelers and language learners. This web application features a custom Spaced Repetition System (SRS) to optimize long-term memory retention.

Key Features:

Dual-Track Learning: tailored modes for Chinese speakers learning Japanese (JLPT N5-N1) and Japanese speakers learning Cantonese.

Smart Review Algorithm: The SRS logic schedules reviews based on your performance (Again, Hard, Good, Easy), utilizing intervals from 10 minutes to several days.

User-Friendly Interface: Includes phonetic guides (Katakana with tone arrows for Cantonese), TTS audio support, and a festive visual theme.

Offline Capable: All progress is saved locally in your browser.
